# BIOS0032_utils

Simple python library with utilities to support the notebooks of the module BIOS0032: AI for the Environment, which is part of the MSc in Ecology and Data Science at University College London (UCL).

## Installation

This package is published on PyPI and can be installed via pip:

```bash
pip install bios0032utils
```

## Features

This library contains utilities for:

*   **Bioacoustics**: Audio processing, evaluation, and plotting tools (including BatDetect2 support).
*   **Vision**: Utilities for classification, detection, and dataset management.

## Module Repository

See the [module repository](https://github.com/MScEcologyAndDataScienceUCL/BIOS0032_AI4Environment) to have a look at the notebooks.
